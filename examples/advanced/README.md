# Advanced Example

This example demonstrates advanced usage of `@gfmio/template-typescript-library`.

## Running the Example

From the repository root:

```bash
# Run the example
task examples:advanced
```

Or from this directory:

```bash
# Run directly
bun run index.ts

# Or via task
task
```

## What This Example Shows

- Importing and using the `greet()` function
- Creating and using a `Calculator` instance
- Basic TypeScript usage with the library
