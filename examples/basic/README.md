# Basic Example

This example demonstrates basic usage of `@gfmio/template-typescript-library`.

## Running the Example

From the repository root:

```bash
# Run the example
task examples:basic
```

Or from this directory:

```bash
# Run directly
bun run start

# Or via task
task
```

## What This Example Shows

- Importing and using the `greet()` function
- Creating and using a `Calculator` instance
- Basic TypeScript usage with the library
