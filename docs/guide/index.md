# Introduction

Welcome to the Template TypeScript Library documentation!

This template provides a comprehensive foundation for building TypeScript libraries with best practices and modern tooling.

## Features

- **TypeScript** - Type-safe development
- **tsup** - Fast bundling with CJS and ESM support
- **Vitest** - Modern, fast testing framework
- **Biome** - Fast linting and formatting
- **ESLint** - JSDoc linting for documentation
- **Prettier** - YAML and Markdown formatting
- **VitePress** - Beautiful documentation site
- **TypeDoc** - Automatic API documentation generation
- **Bun** - Fast package manager
- **Task** - Task automation
- **GitHub Actions** - Automated CI/CD pipeline

## Quick Start

```bash
# Install dependencies
bun install

# Run tests
bun test

# Build library
bun run build

# Start documentation
bun run docs:vitepress:dev
```
